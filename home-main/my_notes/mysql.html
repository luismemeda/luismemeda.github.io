MySQL使用备忘

一些MySQL的使用方法及技巧。俗话说“好记性不如烂笔头”，记下来学而时习之，以防遗忘。

1.限制结果条目——limit
select * from temp limit 3,4 order by ...;
--从第四行开始取四行结果

2.递归CTE（公共表表达式）
with recursive temp as (
select 1 as id1, 1 as value
union all
select id1 + 1 as id111, id1 from temp where id1 < 1000)
select * from temp;
--从1开始生成1000个递增行
